    • What is Electronic Components ?

    There are numerous basic electronic components that are used for building electronic circuits. Without these components, circuit designs are never complete or didnâ€™t function well. These components include resistors, diodes, capacitors, integrated circuits, and so on. Some of these components consists of two or more terminals which are soldered to circuit boards. Some may be packaged type like integrated circuits in which different semiconductor devices are integrated. Here is a brief overview on each of these basic electronic components and you can get in depth information by clicking links attached to each component.

    It is mainly two types :

    1) Passive Components

    A passive element is an electrical component that does not generate power, but instead dissipates, stores, and/or releases it. Passive elements include resistances, capacitors, and coils (also called inductors).


    2) Active Components

    An active component is an electronic component which supplies energy to a circuit. Common examples of active components include: Voltage sources. Current sources.
